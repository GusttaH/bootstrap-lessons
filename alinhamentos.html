Alinhamentos com bootstrap

Uma das coisas mais importantes no desenvolvimento web em geral é dominar bem como alinhar itens de maneira simples horizontalmente ou verticalmente.

Utilizando o bootstrap, nós temos N formas de obter um mesmo resultado de alinhamento, porém é importante buscar o jeito mais simples.

Digamos que nós queiramos centralizar items horizontalmente de maneira rápida utilizando bootstrap, qual caminho seguir?

Exemplo de alinhamentos Horizontais

Os jeitos mais comuns de se alinhar conteúdos é utilizando a margem ou utilizando flex layout.

Utilizando margem

width: 100%

33%
33% - .mx-auto
33% - .ml-auto
Utilizando flex layout

width: 100%

.d-flex .justify-content-start
33%
.d-flex .justify-content-center
33%
.d-flex .justify-content-end
33%

Exemplo de alinhamentos Verticais

Utilizando margem

Quando se trata de alinhamento vertical, temos alguns impasses utilizando a margem, pois ela só alinha automaticamente no centro; além disso também é necessário utilizar ou o display: flex ou o display: grid para que o alinhamento funcione.

width: 100% / height: 200px

.w-100 .h-100 .d-flex

.mx-auto .my-auto

Nota: Todo o retangulo vermelho tem uma altura (height) de 100% baseado no container pai de 200px menos a altura do paragrafo. { height: calc(100% - 23px) }.

Utilizando flex layout

width: 100%

.d-flex .align-items-start
Content Start
.d-flex .align-items-center
Content Center
.d-flex .align-items-end
Content End
